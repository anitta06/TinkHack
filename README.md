# TinkHack 2025 Landing Page

Welcome to the official repository for **TinkHack 2025**, the flagship annual hackathon event organized by TinkerHub MEC. This project contains the source code for the event's landing page, designed to inform and engage participants and visitors.

---

## Motivation

TinkHack provides a platform for coders, designers, and innovators to come together and create breakthrough projects in a fast-paced, collaborative environment. This landing page serves as the starting point for participants to learn about event details, register, and stay connected.

---

## Features

- Responsive and mobile-friendly design.
- Dynamic navigation menu to access sections like Event Overview, Timeline, Registration, Sponsors, and Contact.
- Loading animation with progress bar for an engaging user experience.
- Integration with Google Fonts (Orbitron and Poppins) for modern typography.
- Clear call-to-action buttons and external links for app downloads and social media.

## Usage

To run the landing page locally:

1. Clone this repository.  
2. Open `index.html` in any modern web browser.  
3. Modify `styles.css` and `script.js` as needed to customize the design and interactivity.  
4. Update event dates, sponsor logos, and contact info within the HTML files to keep content current.

---

## File Structure

- `index.html` — Main landing page markup  
- `styles.css` — Stylesheet for visual appearance  
- `script.js` — JavaScript for page interactions and navigation
